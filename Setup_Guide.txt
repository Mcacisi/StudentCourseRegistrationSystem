
THIS IS A BASIC JAVA APPLICATION WHICH CONNECTS TO XAMPP SERVER TO FULLY OPERATE


1.DOWNLOAD JAVA v1.8.0_251 & MYSQL
2.DOWNLOAD JAVA JDK & NETBEANS APACHE 19
3.DOWNLOAD XAMPP SERVER


XAMPP SERVER DB CREDENTIALS:
-if you already have xampp and use your own password & username other than default settings,
 Navigate to the studentDA.java class and change the hardcoded credentials to match yours.


4.START XAMPP SERVER (APACHE & MYSQL),SHOULD DISPLAY GREEN LIGHT IF RUNNING
5.IN PROJECT ROOT, CREATE A FOLDER AND NAME IT lib (SMALL CASE)

  -NAVIGATE TO https://dev.mysql.com/downloads/connector/j/ & DOWNLOAD THE MYSQL J CONNECTOR.JAR FILE AND INSERT IT INTO lib FOLDER

  -NAVIGATE TO NETBEANS ON LEFT PANEL, 
  -GO TO Libraries AND RIGHT CLICK,
  -NAVIGATE TO ADD JAR/FOLDER AND GO TO lib FOLDER YOU CREATED.


5.RUN APPLICATION ON NETBEANS
